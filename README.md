## Intro
這是一個利用 Vue 和 Echarts 製作的 Dashboard

> #### 安裝&啟動
```
npm install
npm run serve
```

> #### 功能需求
+ Bar Chart
+ Line Chart
+ Pie Chart (Doughnut Chart)
+ Weather Panel with OpenWeatherMap Api
+ Radar Chart
+ Multiline Chart
+ Table from Quasar
+ GEO/Map Chart
